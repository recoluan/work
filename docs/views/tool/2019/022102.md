{

      "**/.DS_Store": true,
      "**/tmp": true,

    },
    "files.watcherExclude": {
        "**/.git/objects/**": true,
        "**/.git/subtree-cache/**": true,
        "**/node_modules/**": true,
        "**/tmp/**": true,

      "javascript",

## 快捷键

| 快捷键      | 含义                             |
| ----------- | -------------------------------- |
| alt         | 选中多个光标（需单击，不可拖拽） |
| alt + shift | 选中多个光标（可拖拽）           |
