:::

## 介绍

1. el-scrollbar的父层要有固定高度

2. el-scrollbar的高度要设成100%

3. 如果出现横滚动条，请添加css（.el-scrollbar__wrap{overflow-x:hidden;}）

## Attribute

1. native: Boolean
2. wrapStyle: {} | 子层style
3. wrapClass: {} | 子层class
4. viewClass:{} |  view层class
5. viewStyle:{} |  view层style
6. noresize:Boolean | 是否刷新滚动条大小
