## 查看分支

### 1. 查看本地分支

```bash
git branch
```

### 2. 查看远程分支列表

```bash
git branch -r
```

### 3. 查看所有分支

```bash
git branch -a
```

## 新建分支

### 1. 新建分支

```bash
git branch [branchName]
```

### 2. 切换到另一个分支

```bash
git checkout [branchName]
```

### 3. 新开分支并切换到新分支

```bash
git checkout -b [branchName]
```

## 提交代码

### 1. 向远程仓库提交本地新开的分支

```bash
git push origin [branchName]
```

## 删除

### 1. 删除本地分支

```bash
git branch -d [branchName]

# 强制删除
git branch -D [branchName]
```

### 2. 删除远程分支

```bash
git push origin --delete [branchName]
```

### 3. 清理远程已删除本地没删除的分支

```bash
git fetch -p
```

## 技巧用法

### 1. `|grep` 指令：简单的筛选

```bash
# 筛选包含 br 字段的分支
git branch |grep 'br'
```

### 2.`-E` 筛选中可以使用正则语法

```bash
# 筛选 br-xxx 分支，xxx表示数字的分支
git branch |grep -E 'br-\d+'
```

### 3. `|xargs` 表示把前面的筛选作为后面的参数

```bash
# 筛选 br-000a 这类分支，并删除
git branch |grep -E 'br-\d+[a]$' |xargs git branch -D
```

### 4. `-v` `v` 小写，表示筛选反向

```bash
# 删除 master 外的其他分支
git branch |grep -v 'master' |xargs git branch -D
```
